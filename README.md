# ⚙️ AI-Powered Predictive Maintenance System

🚀 A real-time AI system that predicts machine failures using simulated IoT sensor data, built with Machine Learning, Flask API, and Streamlit dashboard.

---

## 🌟 Key Features

* 🔧 Machine Failure Prediction (ML Model)
* 🌐 Flask API for real-time predictions
* 📊 Interactive Streamlit Dashboard
* ⚡ Live IoT Data Simulation
* 🚨 Alert System (visual + sound)
* 🧠 Confidence Score Prediction
* 🏭 Multi-Machine Monitoring
* 🔐 Login Authentication System
* 💾 Downloadable Logs

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Flask
* Streamlit

---

## ⚙️ System Architecture

Sensor Data → ML Model → Flask API → Dashboard → Alerts & Visualization

## ▶️ How to Run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train model
python main.py

# 3. Run API
cd api
python app.py

# 4. Run Dashboard
streamlit run dashboard/dashboard.py
```
